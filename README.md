This project uses the dataset SuperStore Sales from Kaggle: https://www.kaggle.com/datasets/aditisaxena20/superstore-sales-dataset. It contains  data for the Sales of multiple products sold by the store along with subsequent information related to geography, product categories, subcategories, sales, and profit etc.
By carefully examining this dataset, I was able to identify trends and patterns that provide insight into the sales in different countries and regions. An overview of the overall profit and sales was also explored.
